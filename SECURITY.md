# Security Policy

## Supported Versions

Security fixes are applied to the current 1.x release line.

## Reporting a Vulnerability

Do not open a public issue for a suspected vulnerability. Report it through the
private security contact provided in your MemAura account or contracted support
channel. Include the affected SDK version, reproduction steps, and impact.

Do not include API keys, passwords, session tokens, customer data, or production
logs in the report. Redact sensitive values and provide a minimal reproduction.

We will acknowledge the report, assess the impact, and coordinate a fix through
a private channel.
