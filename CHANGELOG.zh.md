# 更新日志

这里记录所有面向用户的重要变更。

## 2.0.0 - 未发布

### 变更

- 公开 SDK 聚焦于对话写入、job 等待、兼容检索和设备级混合检索。
- `search()` 与 `search_hybrid()` 支持可选的 `session_id` 对话组过滤。
- 支持的运行时矩阵新增 Python 3.13。

### 移除

- 未生效的 `user_id`、公开检查辅助对象、会话状态与 cursor 同步接口。
- 没有受支持公开路由的 Graph/TKG 模型和方法、后端配置辅助对象及原始检查字段。
- 内部传输 client 和 wire type 的顶层导出。
